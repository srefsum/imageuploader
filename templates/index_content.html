How it works

Upload — Select a destination directory and file. Files are saved with sanitized names to prevent path traversal.

Browse — Gallery pages list images as thumbnails. Click an image to view the full file.

Subdirectories — The images directory supports subfolders. You can create new folders from the gallery page.

Serve — Images are served directly via /serve?directory=<key>&file=<name> with MIME detection and caching.

Operational constraints

  • Allowed directories: images, uploads, logos
  • Maximum upload size: 20 MB per request
  • Supported image extensions: .jpg, .jpeg, .png, .gif, .webp
  • Filenames are sanitized to alphanumeric characters, dots, underscores, and hyphens

Security notice

This service has no authentication enabled. Directory access is restricted by an allowlist of path keys. Do not expose this server to untrusted networks without adding access controls.
